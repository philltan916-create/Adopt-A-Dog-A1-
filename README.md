Adopt A Dog Website

Overview
Adopt A Dog is a multi page website is made for a dog adoption agency based in Seattle. The site
allows users to browse through dogs that they possibly will want. as well as be able to process what it would be like to go through the checkout process online.

What the page includes 

Home Page:
- Featured dogs displayed in a grid
- Brief intro to the adoptions agency

Dog Page:
- Full list of available dogs
- Card-based layout that includes costs

Blog Page:
- Info about the dog care
- layout with images and text side by side

Checkout Page:
- Summary of the selected dogs you are adopting
- Total cost of adoption.

Language I used:
- HTML
- CSS

- Overall make the website better for mobile devices and non PC devices
